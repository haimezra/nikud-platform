## מה יש בתיקייה הזו

```
nikud/
├── pages/
│   ├── index.html              ← דף הבית (landing page)
│   ├── login.html               ← כניסה / הרשמה
│   ├── student-dashboard.html   ← דשבורד תלמיד
│   └── teacher-dashboard.html   ← דשבורד מורה (אישור נקודות)
├── lib/
│   ├── schema.sql                ← מבנה מסד הנתונים (להרצה ב-Supabase)
│   └── supabase.js               ← פונקציות עזר (לשימוש עתידי עם React/Next.js)
└── .env.example                  ← תבנית למשתני סביבה
```

---

## שלב 1: הקמת Supabase (15 דקות, חינם)

1. היכנס ל-[supabase.com](https://supabase.com) ופתח חשבון חינמי
2. צור פרויקט חדש (New Project) – תן לו שם כמו "nikud-prod"
3. חכה כ-2 דקות שהפרויקט יוקם
4. לך ל-**SQL Editor** בתפריט הצד
5. פתח את הקובץ `lib/schema.sql` מהתיקייה הזו, העתק את כל התוכן, הדבק ב-SQL Editor ולחץ **Run**
   - זה ייצור את כל הטבלאות (users, schools, lessons, point_logs, store_items, purchases)
   - זה גם יזריע 6 פריטי חנות התחלתיים

6. לך ל-**Settings → API** ושמור 2 ערכים:
   - `Project URL` (משהו כמו `https://xxxxx.supabase.co`)
   - `anon public key` (מחרוזת ארוכה שמתחילה ב-`eyJ...`)

## שלב 2: חיבור הקבצים ל-Supabase

בכל אחד מהקבצים הבאים, חפש את השורות האלו בתחילת ה-`<script>`:

```js
const SUPABASE_URL = 'YOUR_SUPABASE_URL'
const SUPABASE_KEY = 'YOUR_SUPABASE_ANON_KEY'
```

והחלף אותן בערכים האמיתיים שקיבלת בשלב 1:

- `pages/login.html`
- `pages/student-dashboard.html`
- `pages/teacher-dashboard.html`

**טיפ:** עשה חיפוש-והחלפה (Find & Replace) בכל הקבצים בבת אחת.

## שלב 3: בדיקה מקומית (לפני העלאה)

1. פתח את `pages/index.html` בדפדפן (לחיצה כפולה על הקובץ)
2. לחץ "כניסה / הרשמה"
3. נסה להירשם כ**מנהל בית ספר** קודם – זה ייצור בית ספר חדש במסד הנתונים
4. העתק את ה-`school_id` שנוצר (תראה אותו בטבלת `schools` ב-Supabase)
5. הירשם כ**מורה** ואז כ**תלמיד** עם אותו קוד בית ספר

---

## שלב 4: העלאה לאוויר עם הדומיין שלך

### האפשרות הקלה ביותר – Vercel (חינם)

1. היכנס ל-[vercel.com](https://vercel.com) והתחבר עם GitHub
2. העלה את תיקיית `pages` ל-repository חדש ב-GitHub
3. ב-Vercel: **New Project** → בחר את ה-repository → **Deploy**
4. אחרי שזה עולה: **Settings → Domains** → הוסף את הדומיין שלך
5. Vercel ייתן לך רשומות DNS (CNAME / A record) להוסיף אצל ספק הדומיין שלך

### אפשרות חלופית – Netlify

זהה לחלוטין ל-Vercel, גם חינמי, גם תומך בדומיינים מותאמים אישית.

## שלב 5: מה חסר כדי שזה יהיה מוצר מלא

✅ **כבר בנוי:** דף נחיתה, הרשמה/כניסה, דשבורד תלמיד, דשבורד מורה, מסד נתונים, חנות פרסים

⚠️ **כדאי להוסיף בהמשך:**
- **התראות פוש** – חבר את [OneSignal](https://onesignal.com) (חינם עד 10K משתמשים) כדי לשלוח לתלמיד הודעה כשהוא מקבל נקודות
- **דשבורד מנהל בית ספר** – מסך עם נתוני נוכחות לכל הכיתות (אני יכול לבנות את זה בהמשך)
- **מערכת תשלומים** – לגביית מנוי מבתי ספר, מומלץ [Stripe](https://stripe.com) או [Tranzila](https://tranzila.com) (ישראלי)
- **קוד הזמנה לבית ספר** – כרגע ה-school_id הוא ה"קוד", כדאי להחליף לקוד קצר וידידותי יותר (למשל 6 ספרות)
- **אימות אימייל** – Supabase שולח את זה אוטומטית, אבל כדאי להתאים את תבנית המייל בעברית
